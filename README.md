## Intelliplan OSX setup

`curl -O https://raw.githubusercontent.com/kitofr/osx/master/rakefile.rb && rake`


### What it does
 - Installs xcode
 - Setup default osx settings
 - Installs brews
 - Installs casks
 - Installs my dotfiles
 - Installs zshell
 - Asks and sets computer name
 - Asks and sets git name and email
